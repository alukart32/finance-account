Аутентификация

(1) Пользователю будет предложено войти в систему предоставив имя (логин или email) и пароль.
    Имя пользователя и пароль объединяются в экземпляр класса
    UsernamePasswordAuthenticationToken(экземпляр интерфейса Authentication) после чего
    он передается экземпляру AuthenticationManager для проверки.

(2) В случае если пароль не соответствует имени пользователя будет выброшено исключение
    BadCredentialsException с сообщением “Bad Credentials”.

(3) Если аутентификация прошла успешно возвращает полностью заполненный экземпляр Authentication.

(4) Для пользователя устанавливается контекст безопасности путем вызова метода
    SecurityContextHolder.getContext().setAuthentication(…), куда передается объект который вернул AuthenticationManager.
